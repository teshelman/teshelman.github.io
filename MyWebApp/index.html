About the project:

    This application is a Final Project created for Introduction to Internet Mapping (GEOG 485L) at the University of New Mexico

    This project provides large data-set Flight Delay Analysis of carriers and airports in the United States

    Once the page loads the user has the ablility to zoom and pan to any area they choose. To better help determine the delay rate of an airport, the user can pan the map to overlap with the legend. A tooltip has also been provided which holds all of the statistics and airport data for quicker lookup.

    Also provided are two graphs, one is a line graph that depicts the overall delay trend for each month of the selected year, and an interactive piechart which shows the ratio of delay types that occured in the chosen year
